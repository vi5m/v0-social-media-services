# موقع بيع خدمات التواصل الاجتماعي

موقع احترافي وقوي جداً لبيع خدمات التيك توك والإنستغرام والفيسبوك مع لوحة تحكم أدمن متطورة وتصميم عصري.

## المزايا الرئيسية

### للعملاء:
- صفحة رئيسية جذابة بتصميم عصري مع Dark Mode
- عرض شامل للخدمات المتاحة
- نظام سلة تسوق متقدم
- نظام تسجيل/دخول آمن
- تتبع الطلبات والإحصائيات
- عملات متعددة (ريال سعودي، درهم إماراتي، دولار، ريال عماني)
- طرق دفع متعددة (بطاقة ائتمان، تحويل بنكي، لايك كارد، واتس اب)
- صفحات إضافية (عننا، الشروط والأحكام، التواصل)
- نظام Dark Mode كامل

### لوحة تحكم الأدمن المتقدمة:
- **اسم المستخدم:** admin
- **كلمة المرور:** admin123

#### الميزات:
1. **لوحة معلومات شاملة**
   - إجمالي المبيعات والطلبات
   - الطلبات المعلقة والمكتملة
   - آخر الطلبات
   - رسوم بيانية تفصيلية

2. **إدارة الخدمات الكاملة**
   - إضافة خدمات جديدة مع صور
   - تعديل الخدمات الموجودة
   - حذف الخدمات
   - إدارة الأسعار (أسعار متعددة حسب الكمية)
   - رفع وتعديل الصور

3. **إدارة الطلبات المتقدمة**
   - عرض جميع الطلبات
   - تحديث حالة الطلبات (معلقة، مكتملة، ملغاة)
   - عرض تفاصيل الطلب الكاملة
   - تصفية الطلبات

4. **نظام إدارة طرق الدفع**
   - إدارة بيانات التحويل البنكي (البنك، الحساب، الآيبان)
   - إدارة رموز دفع لايك كارد
   - إدارة بيانات واتس اب
   - رسائل خاصة لكل طريقة دفع

5. **الإعدادات الشاملة**
   - معلومات الموقع
   - الحد الأدنى للطلب
   - الرسائل المخصصة
   - إدارة العملات
   - تحديث بيانات الاتصال

## الصفحات الرئيسية

- `index.html` - الصفحة الرئيسية مع عرض الخدمات
- `admin.html` - لوحة تحكم الأدمن
- `about.html` - صفحة عننا مع معلومات الفريق والقيم
- `terms.html` - الشروط والأحكام الكاملة
- `contact.html` - صفحة التواصل مع نموذج رسائل وأسئلة شائعة

## الملفات التقنية

- `styles.css` - أنماط الموقع الرئيسي
- `app.js` - سكريبت الموقع الرئيسي
- `admin-styles.css` - أنماط لوحة الأدمن المتقدمة
- `admin.js` - سكريبت لوحة الأدمن الكامل
- `performance.js` - تحسين الأداء واختبارات آلية

## المميزات التقنية

### التصميم:
- تصميم عصري احترافي بألوان حديثة مميزة
- Dark Mode كامل في جميع الصفحات
- واجهة ذات استجابة (Responsive) لجميع الأجهزة
- أيقونات FontAwesome احترافية
- تأثيرات حركية سلسة

### الأداء:
- تحميل الصور بطريقة Lazy Loading
- تنظيف البيانات القديمة تلقائياً
- تحسين حجم التخزين
- قياس أداء الموقع

### النظم:
- نظام إشعارات Toast تفاعلي

- نظام مراقبة الأداء
- معالجة الأخطاء الشاملة

## كيفية الاستخدام

### للعملاء:
1. افتح `index.html` في المتصفح
2. تصفح الخدمات المتاحة
3. أضف الخدمات المطلوبة إلى السلة
4. اختر العملة المناسبة
5. انتقل إلى الدفع
6. اختر طريقة الدفع المناسبة
7. أتمم عملية الشراء

### للأدمن:
1. افتح `admin.html` في المتصفح
2. سجل الدخول بـ:
   - **اسم المستخدم:** admin
   - **كلمة المرور:** admin123
3. من لوحة المعلومات:
   - أدِر الخدمات (إضافة، تعديل، حذف)
   - إدارة الطلبات وتحديث حالتها
   - أدِر طرق الدفع والرموز
   - عدّل إعدادات الموقع
   - اطلع على الإحصائيات والتقارير

## التخزين المحلي

جميع البيانات يتم حفظها محلياً في المتصفح باستخدام `localStorage`:
- الخدمات والأسعار
- الطلبات والتفاصيل
- بيانات المستخدمين
- السلة وعناصرها
- طرق الدفع والرموز
- إعدادات الموقع
- Theme الاختيار (Light/Dark)
